Gifos App

Gifos is an app where you can search or download the latest gifs based in GIPHY API. You can also create your own gifs through the native library of the explorer webcam and upload it to the web.

Time: 4 Weeks

Tools: Javascript, HTML, CSS.

Introduction

Challenge

To create an interactive web app to share animated GIFs, implementing the front-end development fundamentals and the use of Javascript and Giphy API, working with asynchronous functions and native useful tools from the explorer such as obtaining images from the webcam.

My Role

I was in charge of the front end development of the UI sketch, bring "code life" to the visual product.

Tools Used

UX Design, UI Design, API REST, Javascript, HTML5, CSS, SCSS, Git & Github, Google fonts, FontAwesome, Giphy API, RecordRTC.

The Solution

View Live View Code

User Centered

The design of the app focuses on delivering a user center experience, where you have a more interactive element that can help you find the gif that you want, or suggest the latest gifs and the trending ones also.

Responsive Design

I developed a consistent design using media queries so any device or explorer could have the best experience.

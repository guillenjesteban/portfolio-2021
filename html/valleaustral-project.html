Valle Austral Wines Website

Valle Austral is a fine wine destined for a select target of people, produced in one of the main wine producers in the world: Mendoza, Argentina, and it is only made for exporting to Brazil.

Time: 1 Week

Tools: Adobe XD, Adobe Illustrator, Adobe Photoshop, Visual Studio, HTML, CSS, Javascript.

Introduction

Challenge

Create the product branding & wine concept to communicate it through user touchpoints.

Conduct UX research of personas and website flow.

Craft the UI design according to branding.

Develop and code the web page for the wine.

My Role

In this project, my role had a 360º view/action responsibility, from branding to product development. I intervened as a UI designer, front end developer and branding master.

Tools Used

Concept Design, Information Architecture, App Flow, User Personas, Branding System, UI Design, Landing Page Development.

The Problem

Valle Austral needed user touchpoints for them to collect information about the product. Also they didn’t have a strong and concrete visual identity and could not communicate clearly.

The Solution

Creating a concept that could describe the wine and connect emotionally with the audience and communicate it through user touchpoints, in this case, the creation of a website.

The Process

Research

I investigated competitive websites of other wines from the region that are also exported to extract significant UI Components and with it create in an atomic way, from really tiny to a big masterpiece, to show how the design language could apply to the user interface without interfering in user experience but adding significance to it. After reviewing different existing websites and audience types I developed recommendations on Information Architecture and User Experience, what the site should have, how it should communicate and how it should interact with the possible clients.

Personas

The Explorer

This person likes the discovering of new things, and in this case, new wines. He's passionate about traveling and connecting with different people.

He navigates the page in order to know more about the wine, where and how it is made, if he can relate to it and see every detail that makes the wine special.

The Art Lover

This client is a sociable person who has a cultivated interest in art and everything around it: foods, drinks, social gatherings.

She likes to know about the region where the wine is made, the varieties and what other products we can offer and if we have a strong and consolidated identity that can separate us from other wines.

The Solution

I developed a cohesive and coherent branding and applied UI design with UX patterns to facilitate the journey of the user.

Landing Page

I turned the entire website into a one big artistry journey. Every aspect of the website was built for communication purposes. Whenever you see a design element, it is related to the branding system to connect emotionally with the user and deliver a great experience.

Concept Creation

The design of the landing page focuses on delivering a user center experience, a fast delivery of information with a clear concept about exploring & art. It represents the color of the branding board and has a cohesive design that respects the brand system. The UX Principles applied make a better and more fluid experience for the user.
